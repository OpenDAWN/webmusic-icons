Overview

'webmusic-icons' is a icon-set of PolymerElements/iron-iconset-svg for Web Music Apps.

Repository

GitHub repository

Usage

1. Prepare 'webmusic-icons.html'

Get 'webmusic-icons.html', here is a link : webmusic-icons.html

2. Install 'iron-iconset-svg'

'iron-iconset-svg' depends on polymer and 'iron-icon'. If you use bower, use the command below. It automatically solve the dependency : 
  > bower install --save PolymerElements/iron-iconset-svg
Otherwise, follow links bellow for manual install :
polymer 1.0
PolymerElements/iron-icon
PolymerElements/iron-iconset-svg

3. Add script & link tags for polymer, iron-icon, iron-iconset-svg and webmusic-icons to your HTML

<!doctype html>
<html>
.
.
<script src="bower_components/webcomponentsjs/webcomponents.min.js"></script>
<link rel="import" href="bower_components/iron-icon/iron-icon.html">
<link rel="import" href="bower_components/iron-iconset-svg/iron-iconset-svg.html">
<link rel="import" href="webmusic-icons.html">
.
.
.
</html>

4. Use Icon

Insert a tag like below. 'iconName' specifies the type of icon : 
<iron-icon icon="webmusic-icons:iconName"></iron-icon>

For example...

<iron-icon icon="webmusic-icons:midi"></iron-icon>

Add style-attribute if you want to change size or color :

<iron-icon icon="webmusic-icons:midi" style="width:100px;height:100px;color:magenta"></iron-icon>
